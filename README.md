# 文件说明

2. `latest_base64.txt` 最新订阅文件, 经过base64编码
3. `old*.txt` 历史版本订阅文件
4. `old*_base64.txt` 历史版本订阅文件, 经过base64编码
5. `subs.txt` 使用的订阅源
6. `all_server.txt` 整合的订阅文件, 用于制作可用订阅

因所有人的网络环境不同, 我不能考虑到所有人的环境, 所以提供 `all_server.txt` 文件, 请自行测速使用
